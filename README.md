# Intergraphica

Intergraphica is a lightweight web application for creating interactive infographics — quickly and effortlessly.

It provides curated color palettes, a focused set of visual elements, and intuitive tools to build clean, visually appealing infographics without design overhead.

---

## Core Concepts

- **Users** create and manage projects  
- **Projects** represent individual infographics and their structure

Each project can be used in three modes:

- **Editor** — create and edit the infographic  
- **Preview / Show** — view and share an interactive version via link  
- **Export** — generate a static version for sharing (PDF)

---

## Features

- Design-first infographic editor  
- Predefined color palettes and typography  
- Visual elements (text, cards, shapes, arrows, etc.)  
- Interactive web view (hover, focus, highlights)  
- Static export to **PDF**

---

## Tech Stack

- **Frontend:** React, TypeScript  
- **Backend:** Java, Spring Boot  
- **Database:** PostgreSQL  
- **Infrastructure:** AWS
